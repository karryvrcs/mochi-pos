# 使用教程

## 前言

Mochi POS（摆摊记账）是一款为摊主、小店和轻量零售场景设计的离线收银与经营辅助应用，现已支持 iPhone、iPad、Mac，适合希望快速上手、无收银设备的摊主与店主使用。

你可以在手机或平板上完成点单、结账与查看订单记录，在「商品管理」里维护分类、规格、套餐与库存相关数据，在「订单」中查阅历史单据，使用数据统计功能分析经营情况。数据在设备本地持久保存，并通过 iCloud 在您多台设备之间进行数据同步和备份。

应用支持 20+ 种语言和全球货币，支持多门店切换、折扣与支付方式配置、复杂税率规则设置，并可配合蓝牙热敏打印机打印小票。

使用教程正在编写和完善中。最后更新2026年4月12日。

如有任何疑问，请发送邮件至 [support@tutoorapp.com](mailto:support@tutoorapp.com)。

### 查看订单

点击底部菜单栏的「订单」（iPad / Mac 的菜单栏在顶部）。订单一般按下单时间倒序排列，点击某一单可查看具体订单明细、金额与状态。可用横向状态筛选（如全部、准备中、待取货、已完成、已取消等）快速缩小列表；需要更细条件时，可点击右上角的筛选按钮。

### 查看经营数据

在 **iPhone** 上：请先点击底部菜单栏的「**订单**」进入订单页面，再点击页面右上角的 **图表图标**，即可进入经营数据统计。

在 **iPad / Mac** 上：请点击顶部菜单栏的「**统计**」进入经营数据统计页面。

进入统计页面后，可用日 / 周 / 月 / 年切换视图，点击日期栏的日期可直接切换到某一天或某一段时间区间。统计页面展示营业额、订单量、订单趋势与分类统计等丰富数据，并提供销售报表、每日销售额的数据。

对于免费用户，只提供了今天、昨天、近 7 天的总销售额、总订单数、销售报表、每日销售额四项核心数据。对于付费用户，我们提供了完整的 17 项数据和 11 张图表，并提供周期对比功能。

## 基本功能

### 添加商品

在 **iPhone** 上：点击下方菜单栏的「**商品管理**」，进入商品管理页面。然后点击屏幕右上角的 **+** 号按钮，即可选择添加单品和套餐（在 **1.0.2** 版本后支持添加套餐）。在这里，我们先演示如何添加单品。

在 **iPad / Mac** 上：点击上方菜单栏的「**商品管理**」，进入商品管理页面。然后点击屏幕左侧商品列表中右上角的 **+** 号按钮，即可选择添加单品和套餐（在 **1.0.2** 版本后支持添加套餐）。在这里，我们先演示如何添加单品。

### 编辑商品

在 **iPhone** 上：点击下方菜单栏的「**商品管理**」，进入商品管理页面。然后点击需要修改的商品，即可进入编辑商品页面。

在 **iPad / Mac** 上：点击上方菜单栏的「**商品管理**」，进入商品管理页面。然后点击屏幕左侧商品列表中需要修改的商品，即可进入编辑商品页面。

### 删除商品

在 **iPhone** 上：点击下方菜单栏的「**商品管理**」，进入商品管理页面。左划需要删除的商品，就会显示删除商品按钮，点击该按钮后即可删除商品。如果您只是希望暂时下架商品，应通过编辑商品页面下架商品，而不是删除商品。

在 **iPad / Mac** 上：点击上方菜单栏的「**商品管理**」，进入商品管理页面。在屏幕左侧商品列表中，左划需要删除的商品，就会显示删除商品按钮，点击该按钮后即可删除商品。如果您只是希望暂时下架商品，应通过编辑商品页面下架商品，而不是删除商品。
